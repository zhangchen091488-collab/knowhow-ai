## 🚀 快速开始

### 第一步：了解 Cursor

Cursor 是一个基于 VSCode 的 AI 代码编辑器，核心理念是"AI 不是助手，是 Co-pilot"。

```bash
# 代码补全 - 输入时 AI 自动建议
# 按 Tab 接受，Esc 拒绝

# 代码对话 - Cmd+L 选中代码后提问
# 项目提问 - Cmd+K 引用整个项目
```

### 第二步：掌握核心概念

| 概念 | 说明 | 文档 |
|------|------|------|
| **Rule** | 全局 AI 行为约束 | 入门即掌握 |
| **Command** | 快捷指令 | 入门即掌握 |
| **Skills** | 可复用 AI 能力模块 | [[02-Skills深度探索]] |
| **Subagent** | 多 Agent 协作 | [[05-Subagent协作模式]] |

### 第三步：实践进阶

从简单任务开始，逐步探索高级功能：

```
Week 1: 入门 — 用 AI 完成日常编码任务
Week 2: 进阶 — 用 Rule 规范项目 AI 行为
Week 3: 精通 — 自定义 Command 提升效率
Week 4+: 专家 — Skills 和 Subagent 处理复杂场景
```
