## ⚡ 快速上手

### 方式一：与 Claude Code 配合（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/msitarzewski/agency-agents.git

# 2. 复制 agents 到 Claude Code 目录
cp -r agency-agents/* ~/.claude/agents/

# 3. 在对话中激活 agent
"Hey Claude, activate Frontend Developer mode and help me build a React component"
```

### 方式二：与其他工具集成

```bash
# 一步生成集成文件
./scripts/convert.sh

# 交互式安装（自动检测已安装的工具）
./scripts/install.sh

# 或指定工具
./scripts/install.sh --tool cursor    # Cursor
./scripts/install.sh --tool openclaw # OpenClaw
./scripts/install.sh --tool windsurf  # Windsurf
./scripts/install.sh --tool aider     # Aider
```

### 支持的工具

| 工具 | 集成方式 |
|------|----------|
| Claude Code | `~/.claude/agents/` |
| Cursor | `.cursor/rules/` (.mdc 文件) |
| OpenClaw | `~/.openclaw/agency-agents/` |
| OpenCode | `.opencode/agents/` |
| Aider | `CONVENTIONS.md` |
| Windsurf | `.windsurfrules` |
| Qwen Code | `.qwen/agents/` |

## 🔥 为什么它这么火？

### 1. 极致专业化
不是泛泛的"你是一个程序员"，而是**真正懂行的专家**。每个 Agent 都有：
- 明确的任务边界
- 专业的交付标准
- 独特的工作流程

### 2. 真正的"团队协作"
你可以一次激活多个 Agent：
- 前端 + UI 设计师 = 完美 UI
- 后端 + 安全工程师 = 安全 API
- 营销 + 内容创作者 = 病毒式传播

### 3. 持续进化
- 53K+ Stars
- 7.9K+ Forks
- 社区持续贡献新 Agent
- 每月更新

### 4. 开源免费
