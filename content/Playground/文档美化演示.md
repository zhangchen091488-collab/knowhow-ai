```python
# Python 示例
def calculate_fibonacci(n):
    """计算斐波那契数列"""
    if n <= 1:
        return n
    return calculate_fibonacci(n-1) + calculate_fibonacci(n-2)

# 使用列表推导式
fib_list = [calculate_fibonacci(i) for i in range(10)]
print(fib_list)
```
